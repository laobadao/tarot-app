# TabBar 图标

## 规格

- **尺寸**: 81 x 81 px
- **格式**: PNG
- **透明背景**: 是

## 文件列表

| 文件名 | 说明 |
|--------|------|
| `tab_home.png` | 首页默认图标 |
| `tab_home_active.png` | 首页选中图标 |
| `tab_history.png` | 历史默认图标 |
| `tab_history_active.png` | 历史选中图标 |

## 设计风格

- 线条风格图标
- 金色 (#d4af37) 为选中状态主色
- 灰色为默认状态
- 保持简洁，符合神秘主题
